# GitHub Profile Setup

This folder is set up for a polished GitHub profile repository.

## Publish It

1. Create a public GitHub repository named exactly `aaravk7k`.
2. Put these files in that repository.
3. Update these placeholders in `README.md`:
   - `your.email@example.com`
   - `https://www.linkedin.com/in/your-linkedin`
   - the three featured projects
   - the tech stack, if any badge does not match your actual experience
4. Commit and push.
5. Open the repository's **Actions** tab and run **Generate contribution snake** once.

GitHub automatically shows the `README.md` from `aaravk7k/aaravk7k` at the top of your profile page.

## Hiring Polish Checklist

- Pin your three strongest repositories on your GitHub profile.
- Make sure each pinned repo has a screenshot, setup instructions, and a short "What I learned" section.
- Keep commit messages readable and specific.
- Put your resume, LinkedIn, and best contact method in the profile README.
- Remove or archive unfinished repositories that distract from your best work.
